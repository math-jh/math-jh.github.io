A presheaf contains all the local information, but since there is no way to glue it together to obtain global information, one cannot expect a local-to-global property. A sheaf means those among presheaves that satisfy a gluing condition such as [§Presheaves, ⁋Lemma 1](/en/math/topology/presheaves#lem1).

<div class="definition" markdown="1">

<ins id="def1">**Definition 1**</ins> A presheaf $$\mathscr{F}$$ defined on a topological space $$X$$ is called a *sheaf* if the following two conditions hold.

1. (Identity axiom) For any open set $$U$$ and any open covering $$\{U_i\}_{i\in I}$$, if $$s,t\in\mathscr{F}(U)$$ satisfy $$s\vert_{U_i}=t\vert_{U_i}$$ for all $$i\in I$$, then $$s=t$$.
2. (Gluability axiom) For any open set $$U$$ and any open covering $$\{U_i\}_{i\in I}$$, suppose $$s_i\in\mathscr{F}(U_i)$$ are given and they satisfy $$s_i\vert_{U_i\cap U_j}=s_j\vert_{U_i\cap U_j}$$ for all $$i,j$$. Then there exists some $$s\in \mathscr{F}(U)$$ such that $$s\vert_{U_i}=s_i$$ for all $$i$$.

</div>

For example, by [§Presheaves, ⁋Lemma 1](/en/math/topology/presheaves#lem1), the presheaf of continuous functions from $$X$$ to $$Y$$ defined on a topological space $$X$$ is a sheaf. ([§Presheaves, ⁋Example 3](/en/math/topology/presheaves#ex3))

For a fixed topological space $$X$$, the category $$\Sh(X,\mathcal{A})$$ of $$\mathcal{A}$$-valued sheaves is defined as the full subcategory of $$\PSh(X,\mathcal{A})$$ whose objects are sheaves. That is, the inclusion functor $$\Sh(X, \mathcal{A})\hookrightarrow \PSh(X,\mathcal{A})$$ is the forgetful functor.

Now let us look at the stalk of a sheaf.

<div class="proposition" markdown="1">

<ins id="prop2">**Proposition 2**</ins> For a topological space $$X$$, a sheaf $$\mathscr{F}$$ defined on it, and any open set $$U$$ of $$X$$, the following function

$$\mathscr{F}(U) \rightarrow \prod_{x\in U} \mathscr{F}_x$$

is injective.

</div>
<details class="proof" markdown="1">
<summary>Proof</summary>

The given function is $$s\mapsto (s_x)_{x\in U}$$. Thus it suffices to show that if two $$s,t\in \mathscr{F}(U)$$ satisfy $$s_x=t_x$$ for all $$x\in U$$, then $$s=t$$. 

From the description of the stalk $$\mathscr{F}_x$$ that we examined in [§Presheaves, §§Stalks](/en/math/topology/presheaves#줄기), we know that for each $$x\in U$$ there exists an open set $$x\in V_x\subseteq U$$ such that $$s\vert_{V_x}=t\vert_{V_x}$$. The collection of such $$V_x$$ forms an open covering of $$U$$, and hence we know $$s=t$$ from the first condition of [Definition 1](#def1). 

</details>

If we let $$\mathscr{F}$$ be the sheaf of continuous functions from $$X$$ to $$Y$$, the above proposition roughly means that two continuous functions with the same values at every point are equal. On the other hand, since a function obtained by arbitrarily specifying values at each point is obviously not necessarily continuous, it is obvious that the above function is not surjective. 

<div class="definition" markdown="1">

<ins id="def3">**Definition 3**</ins> For a topological space $$X$$, a sheaf $$\mathscr{F}$$ defined on it, and any open set $$U$$ of $$X$$, we say that an element $$(s_x)_{x\in U}$$ of the set $$\prod_{x\in U} \mathscr{F}_x$$ consists of compatible germs if for each $$x\in U$$ there exists a suitable open neighborhood $$V_x\subseteq U$$ such that, writing $$s_x=[(\tilde{s}_x, V_x)]$$, the following equation

$$(\tilde{s}_x)_y=s_y\in \mathscr{F}_y$$

holds for all $$y\in V_x$$.

</div>

If $$(s_x)_{x\in U}$$ consists of compatible germs, then for any $$x_1,x_2\in U$$ the two elements 

$$\tilde{s}_{x_1}\vert_{V_{x_1}\cap V_{x_2}},\quad \tilde{s}_{x_2}\vert_{V_{x_1}\cap V_{x_2}}$$

of $$\mathscr{F}(V_{x_1}\cap V_{x_2})$$ become the same element by [Proposition 2](#prop2). Hence, using the second condition of [Definition 1](#def1), we can find an appropriate $$\tilde{s}\in \mathscr{F}(U)$$ whose stalk at this element is $$s_x$$.

Then for a presheaf $$\mathscr{F}$$ and a sheaf $$\mathscr{G}$$ defined on $$X$$, a presheaf morphism $$\phi$$ from $$\mathscr{F}$$ to $$\mathscr{G}$$ is completely determined by the morphisms $$\phi_x: \mathscr{F}_x \rightarrow \mathscr{G}_x$$ at each point $$x$$. Moreover, the following holds.

<div class="proposition" markdown="1">

<ins id="prop4">**Proposition 4**</ins> Let a morphism $$\phi:\mathscr{F}\rightarrow \mathscr{G}$$ between sheaves defined on $$X$$ be given. Then $$\phi$$ being an isomorphism is equivalent to all the morphisms $$\phi_x:\mathscr{F}_x \rightarrow \mathscr{G}_x$$ between stalks being isomorphisms.

</div>
<details class="proof" markdown="1">
<summary>Proof</summary>

First, if $$\phi$$ is an isomorphism then it is obvious that the $$\phi_x$$ are isomorphisms. Thus it suffices to show the converse direction. That is, we must prove that if all $$\phi_x$$ are isomorphisms, then $$\phi\vert_U$$ is an isomorphism for every open set $$U$$. 

First we prove that $$\phi\vert_U$$ is always injective. Choose $$s\in \mathscr{F}(U)$$ satisfying $$\phi(s)=0$$. Then for all $$p\in U$$,

$$0=(\phi\vert_U)(s)_x=\phi_x(s_x)$$

holds, and since we assumed $$\phi$$ is an isomorphism at the stalk level, $$s_x=0$$ for all $$p$$. That is, for each $$p\in U$$ there exists a suitable open neighborhood $$W\subseteq U$$ such that $$s\vert_W=0$$. Since the $$W$$'s cover $$U$$, the identity axiom implies $$s=0$$.

Now we prove that $$\phi(U)$$ is surjective. To this end, take arbitrary $$t\in \mathscr{G}(U)$$ and we must construct $$s\in \mathscr{F}(U)$$ satisfying $$(\phi\vert_U)(s)=t$$. Since $$\phi$$ is surjective at the stalk level, for each $$p\in U$$ there exist $$s_x$$ satisfying $$\phi_x(s_x)=t_x$$. Then there exists a suitable open neighborhood $$W\subseteq U$$ of $$p$$ such that the germ at $$p\in U$$ of the section $$s$$ defined on it can be made to be $$s_x$$. Now since

$$(\phi\vert_W)(s)_x=\phi_x(s_x)=t_x$$

holds, if necessary we can restrict $$W$$ to a smaller open neighborhood of $$p$$ so that $$(\phi\vert_W)(s)=t\vert_W$$. Our claim is that the sections constructed pointwise in this way satisfy the gluing condition well, so that we can make a section defined over all of $$U$$. This can be proved as follows. Let $$s\in \mathscr{F}(W)$$ be the section obtained by applying the above argument to $$p\in U$$, and let $$s'\in \mathscr{F}(W')$$ be the section obtained by applying the above argument to $$p'\in U$$. Then on $$W\cap W'$$, 

$$(\phi\vert_{W\cap W'})(s\vert_{W\cap W'})=t\vert_{W\cap W'}=(\phi\vert_{W\cap W'})(s'\vert_{W\cap W'})$$

holds, so by the injectivity of $$\phi\vert_{W\cap W'}$$ we have $$s\vert_{W\cap W'}=s'\vert_{W\cap W'}$$. That the section obtained by gluing these $$s$$'s is sent to $$t$$ via $$\phi$$ is again obvious from the identity axiom of $$\mathscr{G}$$.

</details>

## The Abelian Category of Sheaves

Just as with presheaves, in order to do diagram chasing in the category of (abelian) sheaves, the kernel, cokernel, image, etc. of a morphism between sheaves must themselves be sheaves. However, except for the kernel these are generally not sheaves, so new definitions are needed. 

<div class="definition" markdown="1">

<ins id="def5">**Definition 5**</ins> For any topological space $$X$$ and category $$\mathcal{A}$$, the left adjoint of the forgetful functor $$\Sh(X,\mathcal{A})\rightarrow \PSh(X,\mathcal{A})$$ is called the *sheafification functor*, and is denoted by $$(-)^\dagger$$.

</div>

To complete this definition we need the following lemma.

<div class="proposition" markdown="1">

<ins id="lem6">**Lemma 6**</ins> $$(-)^\dagger:\PSh(X,\Set) \rightarrow \Sh(X,\Set)$$ exists.

</div>

The proof of this is somewhat long, but the gist can be summarized as constructing the sheaf of compatible germs from a presheaf $$\mathscr{F}$$. 

<details class="proof--alone" markdown="1">
<summary>Proof of Lemma 6</summary>

As described above, $$\mathscr{F}^\dagger(U)$$ is given by 

$$\mathscr{F}^\dagger(U)=\{(s_x)_{x\in U}:\text{$(s_x)\in\prod \mathscr{F}_x$ consists of compatible germs}\}$$

First, $$\mathscr{F}^\dagger$$ is a sheaf. Let an open covering $$U=\bigcup U_i$$ be given. If $$s\in \mathscr{F}^\dagger(U)$$ satisfies $$s\vert_{U_i}=0$$ for all $$i$$, then $$s=0$$. This is obvious because for arbitrary $$p\in U$$, if $$p\in U_i$$ then

$$s(p)=(s\vert_{U_i})(p)=0$$

The gluability axiom is trivially satisfied because $$\mathscr{F}^\dagger$$ is defined from the start as a presheaf of functions. The presheaf morphism $$\theta:\mathscr{F}\rightarrow \mathscr{F}^\dagger$$ is of course given, for any open set $$U$$, by

$$\theta\vert_U: \mathscr{F}(U) \rightarrow \mathscr{F}^\dagger(U);\qquad s\mapsto (p\mapsto s_x)$$

For the universal property, if an arbitrary morphism $$\phi: \mathscr{F}\rightarrow \mathscr{G}$$ is given, then for each open set $$U$$ we define $$\psi\vert_U$$ by taking a function $$s\in \mathscr{F}^\dagger(U)$$, picking for each point $$p\in U$$ an open set $$V$$ and $$t\in \mathscr{F}(V)$$ satisfying the second condition above, assigning $$(\phi\vert_V)(t)\in \mathscr{G}(V)$$, and then defining it as the section over $$U$$ obtained by gluing these together. 

</details>

Moreover, the above proof still holds if we replace $$\Set$$ with $$\Ab$$ or $$\Ring$$, etc. Also, in all these cases if $$\mathscr{F}$$ was a sheaf to begin with, the morphism $$\mathscr{F}\rightarrow \mathscr{F}^\dagger$$ obtained from [Lemma 6](#lem6) is an isomorphism. 

As can be seen from the proof, for a presheaf $$\mathscr{F}$$ and its sheafification $$\mathscr{F}^\dagger$$, the stalks $$\mathscr{F}_x$$ and $$\mathscr{F}^\dagger_x$$ at any point $$x\in X$$ are isomorphic. This is obvious because the stalk is a colimit and $$(-)^\dagger$$ is a left adjoint functor. ([\[Category Theory\] §Adjoints, ⁋Definition 9](/en/math/category_theory/adjoints#thm9)) Then by definition we can see that $$\mathscr{F}^\dagger$$ is exactly the sheaf of sections of the canonical projection $$\Spe(\mathscr{F})\rightarrow X$$. ([§Presheaves, ⁋Example 3](/en/math/topology/presheaves#ex3))

<div class="definition" markdown="1">

<ins id="def7">**Definition 7**</ins> Let a sheaf $$\mathscr{F}$$ defined on $$X$$ and a sheaf morphism $$\phi:\mathscr{F}\rightarrow \mathscr{G}$$ be given. 

1. A *subsheaf* of $$\mathscr{F}$$ means a sheaf $$\mathscr{F}'$$ such that $$\mathscr{F}'(U)\subseteq \mathscr{F}(U)$$ for each open set $$U$$.
2. The *kernel* of $$\phi$$ is the kernel of $$\phi$$ as a presheaf morphism. If $$\ker\phi=0$$, we call $$\phi$$ *injective*.
3. The *image* of $$\phi$$ means the sheaf $$(U\mapsto \im\phi\vert_U)^\dagger$$ obtained by applying sheafification to the image of $$\phi$$ as a presheaf morphism; when $$\phi$$ is a sheaf morphism, $$\im\phi$$ is understood to be the sheaf obtained in this way (rather than the image presheaf as a presheaf morphism). In this case, $$\phi$$ being *surjective* means $$\im\phi=\mathscr{G}$$.

</div>

In a similar way, we can define cokernels, quotients, and anything else that fails to take values inside the category of sheaves using sheafification, and that these objects satisfy the corresponding universal property in the sheaf category $$\Sh(X,\mathcal{A})$$ can be shown using only the universal property of [Definition 5](#def5) (without any construction like [Lemma 6](#lem6)).

## Sheaves Defined on a Base

Meanwhile, applying [Lemma 6](#lem6), we can show that in order to describe an arbitrary sheaf $$\mathscr{F}$$ defined on a topological space $$X$$, it suffices to know how this sheaf is defined on a base $$\mathcal{B}$$ of the topology. 

<div class="proposition" markdown="1">

<ins id="prop8">**Proposition 8**</ins> Let an arbitrary topological space $$X$$ and a base $$\mathcal{B}$$ for it be given, and suppose that for each $$B\in \mathcal{B}$$ we are given $$\mathscr{F}(B)$$ and restriction maps

$$\rho_{ji}: \mathscr{F}(B_j) \rightarrow \mathscr{F}(B_i)$$

defined whenever $$B_i,B_j\in \mathcal{B}$$ satisfy $$B_i\subseteq B_j$$, and suppose they satisfy the following two conditions:

1. (Identity axiom) For any $$B\in \mathcal{B}$$ and any open covering $$\{B_i\in \mathcal{B}\}_{i\in I}$$, if $$s,t\in\mathscr{F}(B)$$ satisfy $$s\vert_{B_i}=t\vert_{B_i}$$ for all $$i\in I$$, then $$s=t$$.
2. (Gluability axiom) For any $$B\in \mathcal{B}$$ and any open covering $$\{B_i\}_{i\in I}$$, suppose $$s_i\in\mathscr{F}(B_i)$$ are given and they satisfy $$s_i\vert_{B_i\cap B_j}=s_j\vert_{B_i\cap B_j}$$ for all $$i,j$$. Then there exists some $$s\in \mathscr{F}(B)$$ such that $$s\vert_{B_i}=s_i$$ for all $$i$$.

Then $$\mathscr{F}$$ can be uniquely extended to a sheaf over $$X$$. 

</div>
<details class="proof" markdown="1">
<summary>Proof</summary>

First we can set

$$\mathscr{F}_x=\varinjlim_{x\in B\in \mathcal{B}} \mathscr{F}(B)$$

Now for an arbitrary open set $$U$$, if we define

$$\mathscr{F}(U)=\{(s:U \rightarrow \bigcup_{x\in U} \mathscr{F}_x)\mid s(x)\in \mathscr{F}_x,\quad\text{for $x\in U$, there exists $x\in B\subseteq U$ and $t\in \mathscr{F}(B)$ s.t. $s(x)=t_x$ for all $x\in B$}\}$$

then for the same reason as in [Lemma 6](#lem6) this defines a sheaf.

</details>

## Examples of Sheaves

In the previous post we looked at examples of presheaves, some of which are also sheaves. 

<div class="example" markdown="1">

<ins id="ex9">**Example 9**</ins> The following are examples of sheaves.

- Skyscraper sheaf $$(i_x)_\ast A$$ ([§Presheaves, ⁋Example 5](/en/math/topology/presheaves#ex5))
- If $$\mathscr{F}$$ is a sheaf, then the restriction $$\mathscr{F}\vert_U$$ and the pushforward $$f_\ast \mathscr{F}$$ are also sheaves. ([§Presheaves, ⁋Example 8](/en/math/topology/presheaves#ex8))
- The sheafification of the constant presheaf, $$\underline{A}$$ ([§Presheaves, ⁋Example 6](/en/math/topology/presheaves#ex6)); then $$(i_x)_\ast A$$ is the pushforward of the constant sheaf $$\underline{A}=A$$ on $$\{x\}$$ via the inclusion $$i_x:\{p\}\rightarrow X$$.
- Sheaf of sections ([§Presheaves, ⁋Example 3](/en/math/topology/presheaves#ex3))
- If $$\mathscr{G}$$ is a sheaf then $$\mathscr{Hom}(\mathscr{F},\mathscr{G})$$ is also a sheaf. ([§Presheaves, ⁋Example 12](/en/math/topology/presheaves#ex12)) Likewise the product of sheaves is a sheaf. 

</div>

We do not write out proofs for these separately. 

## Inverse Image Sheaf

We previously saw how to restrict a sheaf $$\mathscr{F}$$ to an open subset $$U$$ of $$X$$; using the inverse image sheaf we can extend this to arbitrary subsets. First, since pushforward $$f_\ast$$ also preserves presheaf morphisms, $$f_\ast:\PSh(X)\rightarrow \PSh(Y)$$ is a functor, and regarding this as a functor defined on the category of sheaves, $$f_\ast: \Sh(X) \rightarrow \Sh(Y)$$ is likewise one. 

<div class="definition" markdown="1">

<ins id="def10">**Definition 10**</ins> The left adjoint of $$f_\ast:\Sh(X) \rightarrow \Sh(Y)$$ is called the *inverse image sheaf*, and is denoted by $$f^{-1}$$. 

</div>

Specifically, for a sheaf $$\mathscr{G}$$ defined on $$Y$$, $$f^{-1}\mathscr{G}$$ is given by first considering the presheaf

$$f^{-1}_\text{pre}\mathscr{G}(U)=\varinjlim_{V\supseteq f(U)} \mathscr{G}(V)$$

and then taking the sheafification of this presheaf:

$$f^{-1}\mathscr{G}=(f^{-1}_\text{pre}\mathscr{G})^\dagger=\left(U\mapsto \varinjlim_{V\supset f(U)} \mathscr{G}(V)\right)^\dagger\tag{$\ast$}$$

<div class="proposition" markdown="1">

<ins id="lem11">**Lemma 11**</ins> The $$f^{-1}$$ defined by the equation ($$\ast$$) above satisfies the adjunction of [Definition 10](#def10):

$$\Hom_{\Sh(X)}(f^{-1} \mathscr{G}, \mathscr{F})\cong \Hom_{\Sh(Y)}(\mathscr{G}, f_\ast \mathscr{F})$$

</div>
<details class="proof" markdown="1">
<summary>Proof</summary>

First, by [[Topology] §Sheaves, ⁋Definition 5](/en/math/topology/sheaves#def5), giving a sheaf morphism $$f^{-1}\mathscr{G} \rightarrow \mathscr{F}$$ is the same as giving a presheaf morphism $$f^{-1}_\text{pre} \mathscr{G} \rightarrow \mathscr{F}$$, and this is the same as having morphisms $$f^{-1}_\text{pre}\mathscr{G}(U) \rightarrow \mathscr{F}(U)$$ for every open set $$U$$ of $$X$$ that preserve restriction. On the other hand, a morphism

$$\varinjlim_{V\supseteq f(U)} \mathscr{G}(V) \rightarrow \mathscr{F}(U)$$

is determined by the morphisms

$$\mathscr{G}(V) \rightarrow \mathscr{F}(U), \qquad\text{$V$ open set in $Y$ containing $f(U)$}$$

by the universal property of the direct limit. 

Now suppose a sheaf morphism $$\phi: \mathscr{G} \rightarrow f_\ast \mathscr{F}$$ is given; defining the corresponding $$\phi^\sharp: f^{-1}\mathscr{G} \rightarrow \mathscr{F}$$ is the same as defining the above $$\mathscr{G}(V) \rightarrow \mathscr{F}(U)$$, and this is defined by the formula

$$\mathscr{G}(V)\overset{\phi(V)}{\longrightarrow}f_\ast\mathscr{F}(V)=\mathscr{F}(f^{-1}(V)) \overset{\rho_{f^{-1}(V), U}}{\longrightarrow} \mathscr{F}(U)$$

Conversely, suppose a morphism $$\psi:f^{-1}\mathscr{G} \rightarrow \mathscr{F}$$ is given and let us define $$\psi^\flat: \mathscr{G}\rightarrow f_\ast\mathscr{F}$$. Viewing this as the composition $$f^{-1}_\text{pre} \rightarrow f^{-1}\mathscr{G}$$ followed by $$f^{-1}\mathscr{G} \rightarrow \mathscr{F}$$ to obtain $$f^{-1}_\text{pre} \mathscr{G} \rightarrow \mathscr{F}$$, and evaluating this presheaf morphism at the open set $$f^{-1}(V)$$ of $$X$$ for an arbitrary open set $$V$$ of $$Y$$, we obtain for each $$V$$

$$f^{-1}_\text{pre} \mathscr{G}(V)=\varinjlim_{W\supseteq f(f^{-1}(V))}\mathscr{G}(W) \rightarrow \mathscr{F}(f^{-1}(V))$$

However, since $$f(f^{-1}(V))\subseteq V$$ holds in general, this data includes $$\mathscr{G}(V) \rightarrow \mathscr{F}(f^{-1}(V))$$, and hence $$\psi^\flat:\mathscr{G} \rightarrow f_\ast \mathscr{F}$$ is defined from this. 

Now $$\phi^{\sharp\flat}=\phi$$ and $$\psi^{\flat\sharp}=\psi$$ hold, and that this bijection is natural can be shown by a straightforward computation. 

</details>

Let us look at the following two particularly noteworthy cases.

<div class="example" markdown="1">

<ins id="ex12">**Example 12**</ins> Consider the case where the function $$f$$ is an open embedding $$i:V\hookrightarrow Y$$ of an open subset $$V$$ into $$Y$$. Then by [§Subspaces, ⁋Lemma 2](/en/math/topology/subspaces#lem2), any open subset $$U$$ of $$V$$ is also open in $$Y$$, so

$$i^{-1}_\text{pre} \mathscr{G}(U)=\varinjlim_{W\supseteq i(U)} \mathscr{G}(W)=\mathscr{G}(i(U))$$

and therefore $$i^{-1}_\text{pre} \mathscr{G}$$ is already a sheaf and 

$$i^{-1}\mathscr{G}=i^{-1}_\text{pre}\mathscr{G}=\mathscr{G}\vert_V$$

</div>

<div class="example" markdown="1">

<ins id="ex13">**Example 13**</ins> Consider the inclusion $$i_y:\{y\}\rightarrow Y$$ of a point in $$Y$$. Then the adjunction of [Lemma 11](#lem11) gives

$$\Hom_{\Sh(\{y\})}(i_y^{-1}
