We now define the notion of an $$A$$-algebra. An algebra is a structure obtained by adding a multiplication to an $$A$$-module $$M$$ that is compatible with the additive structure and scalar multiplication of $$M$$; this is the same process as defining a multiplication on an abelian group $$G$$ that is compatible with its addition. However, there are two problems with using [§Rings, ⁋Definition 1](/en/math/algebraic_structures/rings#def1) directly.

First, for a general ring $$A$$, neither $$\lMod{A}$$ nor $$\rMod{A}$$ is a monoidal category. If $$A$$ is a commutative ring, then $$(\lMod{A},\otimes_A, A)$$ becomes a symmetric monoidal category, so this is resolved by assuming that $$A$$ is a commutative ring. ([§Direct Products, Direct Sums, and Tensor Products of Modules, §§Tensor Products of Modules over Commutative Rings](/en/math/algebraic_structures/operations_of_rings#가환환-위에서-정의된-가군의-텐서곱))

Second, there is a historical issue: if we fix a commutative ring $$A$$ as before and call a monoid object $$E$$ in the symmetric monoidal category $$(\lMod{A},\otimes_A,A)$$ an $$A$$-algebra, then the associativity and unit axioms for a monoid object force the multiplication on $$E$$ to always be associative and to have an identity element. Yet many important examples studied so far often fail to satisfy these properties. To include all of them, we must drop associativity and the existence of an identity element, which also means giving up the definition in [§Rings, ⁋Definition 1](/en/math/algebraic_structures/rings#def1) to some extent.

For these reasons, we shall always assume that $$A$$ is a commutative ring, and define an $$A$$-algebra as follows.

<div class="definition" markdown="1">

<ins id="def1">**Definition 1**</ins> If an $$A$$-bilinear map $$\mu:E\times E \rightarrow E$$ is given on an $$A$$-module $$E$$, then we call $$E$$ an *$$A$$-algebra*. Moreover, depending on the properties satisfied by the multiplication $$\mu$$ on $$E$$, we define the following.

- If the multiplication on $$E$$ satisfies the commutative law, we call it a *commutative $$A$$-algebra*.
- If the multiplication on $$E$$ satisfies the associative law, we call it an *associative $$A$$-algebra*.
- If the multiplication on $$E$$ has an identity element, we call it a *unital $$A$$-algebra*.

</div>

Then a monoid object in $$(\lMod{A},\otimes_A,A)$$ is an associative unital $$A$$-algebra.

On the other hand, an associative unital $$A$$-algebra can be thought of as an $$A$$-module $$E$$ equipped with a multiplication, or as a ring $$E$$ equipped with a scalar multiplication. Fix an arbitrary ring homomorphism $$\rho:A \rightarrow E$$ and assume that $$\rho(A)$$ is contained in the center $$Z(E)$$ of $$E$$. Regarding $$E$$ as a module over itself, we can consider the $$A$$-module $$\rho^\ast E$$ defined via the restriction of scalars

$$\rho^\ast:\lMod{E} \rightarrow \lMod{A}$$

Then the multiplication on $$E$$ gives a map $$\mu:\rho^\ast E\times \rho^\ast E \rightarrow \rho^\ast E$$, and

$$\mu(\alpha\cdot x, y)=(\rho(\alpha)x)y=\rho(\alpha)xy$$

Since $$\rho(\alpha)\in Z(E)$$, through this we can verify the identity

$$\alpha\cdot \mu(x,y)=\mu(\alpha\cdot x,y)=\mu(x,\alpha\cdot y)$$

The ring homomorphism $$\rho:A \rightarrow E$$ giving an $$A$$-algebra structure on $$E$$ is called the *structure morphism*.

<div class="definition" markdown="1">

<ins id="def2">**Definition 2**</ins> Let two $$A$$-algebras $$E,E'$$ be given. An $$A$$-module homomorphism $$u: E \rightarrow E'$$ is called an *$$A$$-algebra homomorphism* if the identity

$$u(xy)=u(x)u(y)$$

holds for all $$x,y\in E$$.

</div>

The following proposition is not difficult to show.

<div class="proposition" markdown="1">

<ins id="prop3">**Proposition 3**</ins> The composition of $$A$$-algebra homomorphisms is an $$A$$-algebra homomorphism. A bijective $$A$$-algebra homomorphism is always an isomorphism.

</div>

These data define the category $$\Alg{A}$$ of $$A$$-algebras. We also define the category $$\cAlg{A}$$ of commutative $$A$$-algebras.

## Examples of Algebras

Just as free rings are defined over abelian groups, free algebras are defined over arbitrary $$A$$-modules. This process is exactly the same as the one constructed in [§Rings, ⁋Proposition 4](/en/math/algebraic_structures/rings#prop4); indeed, this is natural, since the only additional thing we need to show is that the multiplication defined here is compatible with the scalar multiplication structure of $$A$$, and this is already guaranteed because we require $$A$$ to be commutative so that the scalar multiplication structure on $$F(M)=\bigoplus_{n\geq 0}M^{\otimes n}$$ is already defined.

That is, the following holds.

<div class="proposition" markdown="1">

<ins id="prop4">**Proposition 4**</ins> For any $$A$$-module $$M$$, defining

$$F(M)=\bigoplus_{n\geq 0} M^{\otimes n}$$

makes $$F$$ the left adjoint of the forgetful functor $$\Alg{A} \rightarrow \lMod{A}$$.

</div>

The $$F(M)$$ obtained from the above proposition uses the addition already present as data on $$M$$, and introduces a new operation, multiplication. Conversely, it is also possible to make a given group $$G$$ into an $$A$$-algebra by using the multiplication defined on it.

<div class="definition" markdown="1">

<ins id="def5">**Definition 5**</ins> Fix a (not necessarily commutative) ring $$A$$ and an arbitrary group $$G$$. Then the *group ring* $$AG$$ is, as a set, the collection of finitely supported functions from $$G$$ to $$A$$, and for two functions $$\alpha:G \rightarrow A$$, $$\beta: G \rightarrow A$$ their operations are given by

$$\alpha+\beta: x\mapsto \alpha(x)+\beta(x),\qquad \alpha\beta:x\mapsto \sum_{uv=x}\alpha(u)\beta(v)$$

If $$A$$ is a (commutative) ring, then $$AG$$ also carries an $$A$$-module structure, so it becomes an $$A$$-algebra. In this case $$AG$$ is called the *group algebra*.

</div>

For each $$x\in G$$, define $$\delta_x: G \rightarrow A$$ by

$$\delta_x(y)=\begin{cases}1&\text{if $x=y$}\\0&\text{if $x\neq y$}\end{cases}$$

Then any element of $$AG$$ can be written as

$$\sum_{x\in G} \alpha_x\delta_x,\qquad\text{$\alpha_x=0$ for all but finitely many $x$}$$

for a family $$\alpha_x\in A$$ indexed by elements of $$G$$. For convenience, let us write the function $$\delta_x$$ as $$x$$; then the multiplication in $$AG$$ can be written as

$$\left(\sum_{x\in G} \alpha_xx\right)\left(\sum_{y\in G} \beta_yy\right)=\sum_{x,y\in G}\alpha_x\beta_yxy=\sum_{z\in G}\left(\sum_{x\in G}\alpha_x\beta_{x^{-1}z}\right)z$$

Then the following holds.

<div class="proposition" markdown="1">

<ins id="prop6">**Proposition 6**</ins> Consider the two functors

$$A-: \Grp \rightarrow \Alg{A},\qquad (-)^\times:\Alg{A} \rightarrow \Grp$$

Here $$(-)^\times$$ is the functor sending an arbitrary $$A$$-algebra $$E$$ to the group of units $$E^\times$$. Then $$A{-}\dashv (-)^\times$$.

</div>
<details class="proof" markdown="1">
<summary>Proof</summary>

That is, for arbitrary group $$G$$ and $$A$$-algebra $$E$$, we must show the isomorphism

$$\Hom_{\Alg{A}}(AG, E)\cong \Hom_\Grp(G, E^\times)$$

First, suppose a group homomorphism $$f:G \rightarrow E^\times$$ is given. Then from this we can define $$\tilde{f}:AG \rightarrow E$$ by

$$\tilde{f}:AG \rightarrow E;\quad \sum_{x\in G} \alpha_x\delta_x\mapsto \sum_{x\in G} \alpha_xf(x)$$

That $$\tilde{f}$$ preserves addition and scalar multiplication is obvious, and for multiplication as well

$$\begin{aligned}\tilde{f}\left(\left(\sum_{x\in G} a_xx\right)\left(\sum_{y\in G} b_yy\right)\right)&=\tilde{f}\left(\sum_{x,y\in G}a_xb_yxy\right)=\tilde{f}\left(\sum_{z\in G}\left(\sum_{x\in G}a_xb_{x^{-1}z}\right)\right)\\&=\sum_{z\in G}\left(\sum_{x\in G}a_xb_{x^{-1}z}\right)f(z)=\sum_{x,y\in G} a_xb_yf(xy)\\&=\left(\left(\sum_{x\in G} a_xf(x)\right)\left(\sum_{y\in G} b_yf(y)\right)\right)=\tilde{f}\left(\sum_{x\in G} a_xx\right)\tilde{f}\left(\sum_{y\in G} b_yy\right)\end{aligned}$$

so the desired identity holds. Conversely, given an arbitrary $$A$$-algebra homomorphism $$u: AG \rightarrow E$$, the function $$\bar{u}: G \rightarrow E$$ defined by

$$\bar{u}: G \rightarrow E;\qquad x\mapsto u(\delta_x)$$

preserves multiplication. Indeed, for arbitrary $$x,y\in G$$,

$$\bar{u}(xy)=u(\delta_{xy})=u(\delta_x\delta_y)=u(\delta_x)u(\delta_y)=\bar{u}(\delta_x)\bar{u}(\delta_y)$$

so this is obvious, and hence the image of $$\bar{u}$$ lies in $$E^\times$$. That these two processes are inverses of each other is obvious.

</details>

The last example is already somewhat familiar.

<div class="definition" markdown="1">

<ins id="def7">**Definition 7**</ins> For a ring $$A$$, the *polynomial algebra* $$A[\x]$$ is, as a set,

$$A[\x]=\{p(\x)=a_n\x^n+\cdots+a_1\x+a_0\mid a_i\in A\}$$

and addition and multiplication are given by polynomial addition and multiplication, while scalar multiplication by $$A$$ is likewise given by multiplication by a constant. More generally, for an arbitrary set $$S$$, $$A[S]$$ can be defined in a similar way.

</div>

In the case of multivariate polynomials $$A[S]$$, for notational convenience it is common to choose an index set $$I$$ with $$\lvert S\rvert=\lvert I\rvert$$ and denote the elements of $$S$$ by $$\x_i$$. Then for arbitrary

$$\alpha=(\alpha_i)_{i\in I}\in \mathbb{N}_{\geq0}^I\qquad\text{finitely supported}$$

we write

$$\x^\alpha=\prod_{i\in I}\x_i^{\alpha_i}$$

and elements of $$A[\x_i]_{i\in I}$$ can be written as

$$\sum_{d=0}^n \sum_{\lvert\alpha\rvert=d} a_\alpha\x^\alpha$$

Here $$\lvert\alpha\rvert=\sum_{i\in I}\alpha_i$$.

Similarly, polynomial algebras also satisfy the following universal property.

<div class="proposition" markdown="1">

<ins id="prop8">**Proposition 8**</ins> The functor $$A[-]:\Set \rightarrow \cAlg{A}$$ sending a set $$S$$ to the polynomial algebra $$A[\x_i]_{i\in I}$$ whose variables are the elements of $$S$$ is the left adjoint of the forgetful functor $$U: \cAlg{A} \rightarrow \Set$$.

</div>
<details class="proof" markdown="1">
<summary>Proof</summary>

That is, we must show the isomorphism

$$\Hom_{\cAlg{A}}(A[\x_i]_{i\in I}, E)\cong\Hom_\Set(S, UE)$$

First, suppose an arbitrary function $$f:S \rightarrow U(E)$$ is given. Then the map $$\tilde{f}: A[\x_i]_{i\in I} \rightarrow E$$ defined by

$$\tilde{f}: \sum_{d=0}^n \sum_{\lvert\alpha\rvert=d} a_\alpha\x^\alpha\mapsto \sum_{d=0}^n\sum_{\lvert\alpha\rvert=d}a_\alpha\prod_{i\in I}f(\x_i)^{\alpha_i}$$

is an $$A$$-algebra homomorphism from $$A[\x_i]_{i\in I}$$ to $$E$$. Conversely, given an arbitrary $$A$$-algebra homomorphism $$u:A[\x_i]_{i\in I} \rightarrow E$$, we define $$\bar{u}:S \rightarrow UE$$ by $$\x_i\mapsto u(\x_i)$$. It is easy to verify that these are inverses of each other.

</details>

Thus polynomial algebras can be thought of as a kind of free commutative $$A$$-algebra. Comparing [Definition 5](#def5) and [Definition 7](#def7) reveals a slight similarity; if we generalize [Definition 5](#def5) to the *monoid ring*, then the polynomial ring in [Definition 7](#def7) becomes the monoid ring formed from the monoid consisting of elements of the form $$\x^\alpha$$.

## Subalgebras, Ideals, and Quotient Algebras

<div class="definition" markdown="1">

<ins id="def9">**Definition 9**</ins> A submodule $$F$$ of an $$A$$-algebra $$E$$ is called a *subalgebra* of $$E$$ if $$F$$ is closed under the multiplication of $$E$$.

</div>

Meanwhile, we also define ideals of an $$A$$-algebra in the same way as we defined them in [§Rings, ⁋Definition 7](/en/math/algebraic_structures/rings#def7).

<div class="definition" markdown="1">

<ins id="def10">**Definition 10**</ins> A subalgebra $$\mathfrak{a}$$ of an $$A$$-algebra $$E$$ is called a *left ideal* of $$E$$ if for arbitrary $$x\in \mathfrak{a}$$ and $$\alpha\in E$$, we have $$\alpha x\in \mathfrak{a}$$. Similarly we also define a *right ideal*. A left ideal that is also a right ideal is called a *two-sided ideal*.

</div>

Thus an ideal of an $$A$$-algebra is nothing special: it is simply the ideal of $$E$$ regarded as a ring, forgetting the $$A$$-action defined on $$E$$. For example, the kernel

$$\ker u=\{x\in E\mid u(x)=0\}$$

of an $$A$$-algebra homomorphism $$u:E \rightarrow F$$ is a two-sided ideal of $$E$$. Then we can define the following.

<div class="definition" markdown="1">

<ins id="def11">**Definition 11**</ins> For an arbitrary $$A$$-algebra $$E$$ and a two-sided ideal $$\mathfrak{a}$$ of $$E$$, we call $$E/\mathfrak{a}$$ the *quotient algebra* of $$E$$ by $$\mathfrak{a}$$.

</div>

Of course, having defined it this way, one should prove that $$E/\mathfrak{a}$$ is an $$A$$-algebra, but this is the same as for rings so we omit it. Also, the following holds.

<div class="proposition" markdown="1">

<ins id="prop12">**Proposition 12**</ins> For an $$A$$-algebra homomorphism $$u:E \rightarrow F$$ and its kernel $$\ker u$$, the following hold.

1. $$\ker u$$ is a two-sided ideal of $$E$$, and $$x+\ker u \mapsto u(x)$$ defines a well-defined isomorphism $$E/\ker u \rightarrow \im u$$.
2. For a subalgebra $$E'$$ of $$E$$, $$E'+\ker u=\{x'+y\mid x'\in E', y\in\ker u\}$$ is a subalgebra of $$E$$, and $$E'\cap\ker u$$ is a two-sided ideal of $$E'$$, and there is an isomorphism $$(E'+\ker u)/\ker u\cong E'/(E'\cap \ker u)$$.
3. If two two-sided ideals $$\mathfrak{a}, \mathfrak{b}$$ of $$E$$ satisfy $$\mathfrak{b}\subseteq \mathfrak{a}$$, then $$\mathfrak{a}/\mathfrak{b}$$ is a two-sided ideal of $$E/\mathfrak{b}$$ and $$(E/\mathfrak{b})/(\mathfrak{a}/\mathfrak{b})\cong E/\mathfrak{a}$$ holds.
4. For a two-sided ideal $$\mathfrak{a}$$ of $$E$$, there exists an inclusion-preserving bijection between the set of two-sided ideals of $$E/\mathfrak{a}$$ and the set of ideals of $$A$$ containing $$\mathfrak{a}$$.

</div>
