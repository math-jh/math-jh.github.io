## Semigroups and Monoids

<div class="definition" markdown="1">

<ins id="def1">**Definition 1**</ins> A magma $$(A, \star)$$ that satisfies the associative law is called a *semigroup*.

</div>

The homomorphisms, substructures, and quotient structures defined for magmas are well-defined for semigroups without any modification. In particular, if $$A$$ is a semigroup, then any submagma $$S$$ of $$A$$ is also a semigroup.

<div class="definition" markdown="1">

<ins id="def2">**Definition 2**</ins> For any magma $$(A,\star)$$, an element $$e\in A$$ satisfying

$$x\star e=e\star x=x$$

for all $$x\in A$$ is called an *identity element*.

</div>

Any magma $$A$$ has at most one identity element. If both $$e$$ and $$e'$$ are identity elements of $$A$$, then

$$e=e\star e'=e'\star e=e'$$

so they must be equal.

On the other hand, any element of a set $$A$$ can be regarded as a function from a singleton $$\ast$$ to $$A$$. ([\[Category Theory\] §Representable Functors, ⁋Example 2](/en/math/category_theory/representable_functors#ex2)) From this perspective, $$e:\ast\rightarrow A$$ is an identity element precisely when the following diagram

![identity](/assets/images/Math/Algebraic_Structures/Groups-1.png){:style="width:16.4em" class="invert" .align-center}

commutes.

<div class="definition" markdown="1">

<ins id="def3">**Definition 3**</ins> A semigroup $$(A,\star)$$ with an identity element is called a *monoid*.

</div>

Since a monoid is well-defined only when we have the set $$A$$, the operation $$\star$$ defined on it, and the identity element $$e$$ with respect to $$\star$$, we denote a monoid by the triple $$(A,\star, e)$$. From the above discussion, we see that a monoid is a monoid object in $$\Set$$. ([[\[Category Theory\] §Monoid Objects, ⁋Definition 1](/en/math/category_theory/monoid_objects#def1))

Care is required when defining *monoid homomorphisms* and *submonoids*. For instance, a magma homomorphism $$f:A\rightarrow A'$$ between two monoids $$(A,\star,e)$$ and $$(A',\star',e')$$ need not preserve the identity element, so a monoid homomorphism is defined to preserve $$e$$ as well.

<div class="definition" markdown="1">

<ins id="def4">**Definition 4**</ins> For two monoids $$(A, \star, e)$$ and $$(A',\star', e')$$, a magma homomorphism satisfying $$f(e)=e'$$ is called a *monoid homomorphism*.

</div>

Monoids and monoid homomorphisms defined in this way form a category.

<div class="proposition" markdown="1">

<ins id="prop5">**Proposition 5**</ins> There exists a category $$\Mon$$ whose objects are monoids and whose morphisms are monoid homomorphisms.

</div>
<details class="proof" markdown="1">
<summary>Proof</summary>

Let $$f:M_1\rightarrow M_2$$ and $$g:M_2\rightarrow M_3$$ be monoid homomorphisms. By [§Algebraic Structures, ⁋Proposition 7](/en/math/algebraic_structures/algebraic_structures#prop7), $$g\circ f$$ is a magma homomorphism. Furthermore, the equation

$$(g\circ f)(e_1)=g(f(e_1))=g(e_2)=e_3$$

shows that $$g\circ f$$ is also a monoid homomorphism.

Since monoid homomorphisms are functions, composition is associative. Moreover, for any monoid $$M$$, the identity map $$\id_M$$ is a monoid homomorphism.

</details>

Moreover, a submagma of a monoid need not contain the identity element, so the following definition is needed.

<div class="definition" markdown="1">

<ins id="def6">**Definition 6**</ins> A *submonoid* of a monoid $$(A,\star, e)$$ is a submagma of $$A$$ that contains the identity element $$e$$.

</div>

However, given a family of submonoids $$(S_i)$$ of a monoid $$(A,\star,e)$$, their intersection $$S=\bigcap S_i$$ is again a submonoid, since $$e\in S_i$$ for all $$i$$ and hence $$e\in S$$.

For quotient structures, given a monoid $$(A, \star,e)$$ and an equivalence relation $$R$$ compatible with $$\star$$, the quotient $$A/R$$ naturally inherits a monoid structure. Indeed, considering the equivalence class $$[e]$$ of $$e$$, for any $$[x]\in A/R$$ we have

$$[x]\mathbin{\tiny\char"2606}[e]=[x\star e]=[x]=[e\star x]=[e]\mathbin{\tiny\char"2606}[x]$$

holds.

Assuming the existence of an identity element on a magma is one of the stronger conditions. For example, the following theorem shows that there is only one way to equip a set $$X$$ with two compatible magma structures with identity elements, and the result is a commutative monoid.

<div class="proposition" markdown="1">

<ins id="thm7">**Theorem 7 (Eckmann-Hilton)**</ins> Let $$\star_1$$ and $$\star_2$$ be two operations on a set $$X$$ such that $$(X,\star_1,e_1)$$ and $$(X,\star_2,e_2)$$ are both magmas with identity elements. If

$$(a\star_1 b)\star_2(c\star_1 d)=(a\star_2 c)\star_1(b\star_2 d)$$

holds for all $$a,b,c,d\in X$$, then $$\star=\star_1=\star_2$$, $$e=e_1=e_2$$, and $$(X,\star,e)$$ is a commutative monoid.

</div>
<details class="proof" markdown="1">
<summary>Proof</summary>

First, we show that $$e_1=e_2$$. This follows from the equation

$$e_1=e_1\star_1 e_1=(e_1\star_2e_2)\star_1(e_2\star_2e_1)=(e_1\star_1 e_2)\star_2(e_2\star_1 e_1)=e_2\star_2 e_2=e_2$$

Now, for any $$a,b$$, we have

$$a\star_1 b=(a\star_2 e_2)\star_1(e_2\star_2b)=(a\star_1 e_2)\star_2(e_2\star_1b)=a\star_2b$$

which shows that $$\star=\star_1=\star_2$$. Furthermore,

$$a\star b=(e\star a)\star(b\star e)=(e\star b)\star(a\star e)=b\star a$$

and

$$a\star(b\star c)=(a\star 1)\star(b\star c)=(a\star b)\star(1\star c)=(a\star b)\star c$$

Therefore, $$(X,\star,e)$$ is a commutative monoid.

</details>

## Groups

Finally, we define groups. Intuitively, a group is a monoid in which every element is invertible.

<div class="definition" markdown="1">

<ins id="def8">**Definition 8**</ins> In a monoid $$(A,\star,e)$$, an element $$x$$ is *left cancellable* if $$x\star a=x\star b$$ implies $$a=b$$ for all $$a,b\in A$$. We define *right cancellable* elements analogously. An element $$x$$ is a *left inverse* of $$y$$ if $$x\star y=e$$; a *right inverse* is defined similarly.

If $$x$$ is both a left inverse and a right inverse of $$y$$, then $$x$$ is called an *inverse* of $$y$$, and in this case $$y$$ is said to be *invertible*.

</div>

In general, a monoid may contain an element that has a left inverse but no right inverse, and conversely, an element that has a right inverse but no left inverse. The inverse of an element $$x$$ is usually denoted by $$x^{-1}$$; if the operation is written as $$+$$, we instead write $$-x$$. For such a symbol to be well defined, the inverse must be uniquely determined.

<div class="proposition" markdown="1">

<ins id="prop9">**Proposition 9**</ins> In a monoid $$(A, \star, e)$$, if an element $$x\in A$$ is invertible, then its inverse is unique.

</div>
<details class="proof" markdown="1">
<summary>Proof</summary>

If $$x'$$ and $$x''$$ are both inverses of $$x$$, then

$$x'=x'\star e=x'\star( x\star x'')=(x'\star x)\star x''=e\star x''=x''$$

hence $$x'=x''$$.

</details>

Using this, we obtain the following corollary.

<div class="proposition" markdown="1">

<ins id="cor10">**Corollary 10**</ins> For invertible elements $$a,b$$ of a monoid $$(A,\star,e)$$, the following hold.

1. $(a^{-1})^{-1}=a$
2. $(a\star b)^{-1}=b^{-1}\star a^{-1}$.

</div>
<details class="proof" markdown="1">
<summary>Proof</summary>

By the previous proposition, inverses are unique; thus it suffices to verify directly that the right-hand sides satisfy the defining conditions for an inverse.

First, we check that $$a$$ is the inverse of $$a^{-1}$$. The inverse of $$a^{-1}$$ is an element $$x$$ satisfying

$$a^{-1}\star x=x\star a^{-1}=e$$

Since

$$a^{-1}\star a=a\star a^{-1}=e$$

holds by the definition of $$a^{-1}$$, $$x=a$$ satisfies the above equations. Since the inverse of $$a^{-1}$$ is unique, the inverse $$(a^{-1})^{-1}$$ must be $$a$$.

Similarly, the second claim follows immediately from the equations

$$\begin{aligned}(a\star b)\star(b^{-1}\star a^{-1})&=a\star(b\star b^{-1})\star a^{-1}=a\star e\star a^{-1}=a\star a^{-1}=e,\\(b^{-1}\star a^{-1})\star(a\star b)&=b^{-1}\star(a^{-1}\star a)\star b=b^{-1}\star e\star b=b^{-1}\star b=e.\end{aligned}$$

</details>

Now we define groups as follows.

<div class="definition" markdown="1">

<ins id="def11">**Definition 11**</ins> A monoid in which every element is invertible is called a *group*. If $$\star$$ is commutative, it is called an *abelian group* (or *commutative group*).

</div>

Taking inverses defines a function from $$G$$ to itself[^1], so a group $$G$$ is determined by the data $$(G,\star,e, (-)^{-1})$$. The inverse $$(-)^{-1}$$ can be represented by the following diagram

![inverse](/assets/images/Math/Algebraic_Structures/Groups-2.png){:style="width:13em" class="invert" .align-center}

From this, we can verify that any group is a group object in $$\Set$$. ([\[Category Theory\] §Monoid Objects, ⁋Definition 1](/en/math/category_theory/monoid_objects#def1))

On the other hand, a monoid homomorphism $$f:G\rightarrow G'$$ must preserve inverses:

$$f(x)\star'f(x^{-1})=f(x\star x^{-1})=f(e)=e',\qquad f(x^{-1})\star'f(x)=f(x^{-1}\star x)=f(e)=e'.$$

Thus $$\Grp$$ is a full subcategory of $$\Mon$$. ([\[Category Theory\] §Functors, ⁋Definition 10](/en/math/category_theory/functors#def10))

Moreover, for a *magma homomorphism* $$f:G\rightarrow G'$$ between two groups, we have

$$e'\star' f(e)=f(e)=f(e\star e)=f(e)\star'f(e)$$

and multiplying on the right by the inverse of $$f(e)$$ yields $$e'=f(e)$$. Thus by the previous argument, $$\Grp$$ is also a full subcategory of $$\Magma$$.

The above argument used the following lemma.

<div class="proposition" markdown="1">

<ins id="lem12">**Lemma 12 (Cancellation law)**</ins> Any invertible element is cancellable.

</div>
<details class="proof" markdown="1">
<summary>Proof</summary>

Multiply both sides on the left or on the right by the inverse of $$a$$.

</details>

By the same argument as in [Proposition 5](#prop5), groups and group homomorphisms also form a category.

<div class="proposition" markdown="1">

<ins id="prop13">**Proposition 13**</ins> There exists a category $$\Grp$$ whose objects are groups and whose morphisms are group homomorphisms. Likewise, there exists a full subcategory $$\Ab$$ whose objects are abelian groups and whose morphisms are group homomorphisms.

</div>

We can verify that these categories have a zero object $$\{e\}$$. We define subgroups similarly to submonoids.

<div class="definition" markdown="1">

<ins id="def14">**Definition 14**</ins> A subset $$S$$ of a group $$(G,\star, e, {}^{-1})$$ is a *subgroup* if $$S$$ is a submonoid that is closed under taking inverses.

</div>

The following proposition provides a single criterion for determining whether a given subset is a subgroup, without checking the existence of an identity element, closure under inverses, and so on.

<div class="proposition" markdown="1">

<ins id="prop15">**Proposition 15**</ins> A nonempty subset $$S$$ of a group $$(G, \star, e, {}^{-1})$$ is a subgroup of $$G$$ if and only if $$a\star b^{-1}\in S$$ for all $$a,b\in S$$.

</div>
<details class="proof" markdown="1">
<summary>Proof</summary>

If $$S$$ is a subgroup of $$G$$, then $$b\in S$$ implies $$b^{-1}\in S$$, and therefore $$a\star b^{-1}\in S$$ trivially.

Thus it suffices to prove the converse. Since $$S$$ is nonempty, there exists some $$a\in S$$, and then $$a\star a^{-1}\in S$$, so $$e\in S$$. For any $$a\in S$$, we have $$a^{-1}=e\star a^{-1}\in S$$. Moreover, for any $$a,b\in S$$, we have $$a\star b^{-1}=a\star(b^{-1})^{-1}\in S$$.

</details>

For a family of subgroups $$(S_i)$$ of a group $$G$$, the intersection $$S=\bigcap S_i$$ is a subgroup. Choosing arbitrary $$a,b\in S$$, we have $$ab^{-1}\in S_i$$ for all $$i$$, and therefore $$ab^{-1}\in S$$. In particular, for any subset $$S$$ of $$G$$, applying this observation to the collection of subgroups of $$G$$ containing $$S$$ yields the smallest subgroup containing $$S$$. This is denoted by $$\langle S\rangle$$. With a little work, one can show that $$\langle S\rangle$$ coincides with the set of all elements obtainable from those of $$S$$ by finitely many operations.

On the other hand, we have already seen that for a group $$(G, \star, e)$$ and an equivalence relation $$R$$ compatible with $$\star$$, the quotient $$G/R$$ has a monoid structure; in fact, it also has a group structure. To see this, it suffices to show that every element $$[x]$$ of $$G/R$$ is invertible. Indeed,

$$[x]\mathbin{\tiny\char"2606}\bigl[x^{-1}\bigr]=\bigl[x\star (x^{-1})\bigr]=[e]=\bigl[x^{-1}\star x\bigr]=\bigl[x^{-1}\bigr]\mathbin{\tiny\char"2606}[x]$$

so every element of $$G/R$$ is invertible.

When dealing with general groups, we always denote the operation as multiplication, abbreviate $$x\star y$$ to $$xy$$, write the inverse of $$x$$ as $$x^{-1}$$, and denote the identity element as $$e$$, as above. If the group $$G$$ is abelian, however, we denote the operation as addition, the inverse of $$x$$ as $$-x$$, and the identity element as $$0$$.
