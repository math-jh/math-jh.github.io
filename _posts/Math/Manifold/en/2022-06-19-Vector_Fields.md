## Vector Fields

We defined vector bundles in the previous post partly for later use, but we also needed them right away to define vector fields.

<div class="definition" markdown="1">

<ins id="def1">**Definition 1**</ins> For any vector bundle $$\pi:E\rightarrow M$$, a map $$\sigma:M\rightarrow E$$ satisfying $$\pi\circ\sigma=\id_M$$ is called a *section* of the vector bundle $$E\rightarrow M$$. 

</div>

This definition merely says that $$\sigma$$ is a function taking each point $$p$$ to an element of $$E_p$$. Meanwhile, since $$E$$ is also a manifold, any section $$\sigma:M\rightarrow E$$ is a map between two manifolds, and thus the smoothness of $$\sigma$$ is well-defined. We write $$\Gamma(E)$$ for the set of smooth sections of the vector bundle $$E\rightarrow M$$. 

In particular, a section $$X$$ of the tangent bundle $$\pi:TM\rightarrow M$$ is called a *vector field*, and the value of $$X$$ at a point $$p$$ is often written as $$X_p$$. Then since $$X_p\in T_pM$$, for a function $$f$$ defined near $$p$$, the real number $$X_p(f)$$ is well-defined. In particular, if $$f$$ is defined on an open set $$U$$, we denote by $$X(f)$$ the function $$U\rightarrow\mathbb{R}$$ that takes $$p\in U$$ to the real number $$X_p(f)$$.

<div class="proposition" markdown="1">

<ins id="prop2">**Proposition 2**</ins> Let $$X$$ be a vector field defined on a manifold $$M$$. Then the following are all equivalent.

1. $$X$$ is $$C^\infty$$.
2. For any coordinate system $$(U,\varphi)$$, $$\varphi=(x^i)_{i=1}^m$$, the functions $$a^i:U\rightarrow\mathbb{R}$$ defined by the formula
    
    $$(X|_U)_p=\sum_{i=1}^m a^i(p)\frac{\partial}{\partial x^i}\bigg|_p$$

    are all $$C^\infty$$ functions defined on $$U$$. 
3. For any open set $$V\subseteq M$$ and any $$C^\infty$$ function $$f:V\rightarrow\mathbb{R}$$ defined on $$V$$, $$X(f)$$ is also $$C^\infty$$.

</div>
<details class="proof" markdown="1">
<summary>Proof</summary>

First, if $$X$$ is $$C^\infty$$, then it is obvious that $$X\vert_U$$ is $$C^\infty$$. Meanwhile, as functions on $$TM$$, the $$dx^i$$ are component functions of the coordinate system $$\tilde{\varphi}:\pi^{-1}(U)\rightarrow\mathbb{R}^{2m}$$, so they are likewise $$C^\infty$$, and therefore their composition

$$a^i(p)=dx^i\left(\sum a^i(p)\frac{\partial}{\partial x^i}\bigg|_p\right)=dx^i|_p\circ(X|_U)_p$$

is also obviously $$C^\infty$$. 

Assume that condition 2 holds. To show that $$X(f)$$ is $$C^\infty$$ at any $$p\in V$$, it suffices to show that $$X(f)$$ is $$C^\infty$$ at $$p$$ for a suitable coordinate system $$(U,\varphi)$$ containing $$p$$. Now by condition 2, 

$$(X|_U)_pf=\sum_{i=1}^m a^i(p)\frac{\partial f}{\partial x^i}(p)$$

and then the right-hand side is a $$C^\infty$$ function defined on $$U$$, so $$X(f)$$ is a $$C^\infty$$ function.

Finally, assume condition 3 and show 1. To show that $$X:M\rightarrow TM$$ is $$C^\infty$$, it suffices to show that for any coordinate system $$(U,\varphi)$$, the map $$X\circ\varphi^{-1}$$ is a $$C^\infty$$ function from $$\varphi(U)$$ to $$TM$$, and considering the coordinate system on $$TM$$, this amounts to showing that the following functions

$$x^i\circ\pi\circ (X|_U),\quad dx^i\circ(X|_U)$$

are $$C^\infty$$. But a direct computation shows that

$$x^i\circ\pi\circ (X|_U)=x^i\circ\id_U=x^i,\qquad dx^i\circ(X|_U)=X(x^i)$$

so they are all $$C^\infty$$.

</details>

We write $$\mathfrak{X}(M)$$ for the set of $$C^\infty$$ vector fields defined on $$M$$, and henceforth assume all vector fields are $$C^\infty$$. Meanwhile, by a partition of unity, a vector field defined only on some open set $$U$$ of $$M$$ can also be extended to all of $$M$$. ([§Tangent Space, footnote 1](/en/math/manifold/tangent_space#fn:1))

## Local Frame

The addition of two elements $$X,Y$$ in the space $$\mathfrak{X}(M)$$ is well-defined as follows.

$$(X+Y)_p=X_p+Y_p$$

Also, for any real number $$\alpha\in\mathbb{R}$$, defining

$$(\alpha X)_p=\alpha\cdot X_p$$

we see that $$\alpha X$$ is also an element of $$\mathfrak{X}(M)$$, so $$\mathfrak{X}(M)$$ becomes an $$\mathbb{R}$$-vector space. However, as an $$\mathbb{R}$$-vector space, $$\mathfrak{X}(M)$$ is too large. 

<div class="example" markdown="1">

<ins id="ex3">**Example 3**</ins> Let $$M=\mathbb{R}$$. Here we think of $$M$$ as being given a manifold structure via the single chart $$(\mathbb{R},\id)$$. Then the tangent space $$T_pM$$ at each point $$p$$ is a one-dimensional vector space generated by $$d/dx\vert_p$$, and thus the correspondence

$$X:M\rightarrow TM;\qquad p\mapsto \frac{d}{dx}\bigg\vert_p$$

is an element of $$\mathfrak{X}(M)$$. However, for any $$C^\infty$$ function $$f:\mathbb{R}\rightarrow\mathbb{R}$$,

$$fX:M\rightarrow TM;\qquad p\mapsto f(p)\frac{d}{dx}\bigg\vert_p$$

also belongs to $$\mathfrak{X}(M)$$ ([Proposition 2](#prop2)), and this element cannot be expressed as a constant multiple of $$X$$ unless $$f$$ is a constant function. Moreover, since $$C^\infty(M)$$ is an infinite-dimensional vector space over $$\mathbb{R}$$, $$\mathfrak{X}(M)$$ is also infinite-dimensional.

</div>

In such a situation, it is relatively more convenient to treat $$\mathfrak{X}(M)$$ as a $$C^\infty(M)$$-module. 

In the above, $$\mathfrak{X}(M)$$ was generated by the vector field $$d/dx$$, but in general, since $$C^\infty(M)$$ is not a field, there can very well be $$C^\infty(M)$$-modules that do not have a basis. For example, the well-known [hairy ball theorem](https://en.wikipedia.org/wiki/Hairy_ball_theorem) shows that any continuous vector field defined on the unit sphere $$S^2$$ in 3-dimensional space must have a point where it vanishes. 

![Hairy_ball](/assets/images/Math/Manifold/Vector_Fields-1.png){:style="width:200px" class="invert" .align-center}

Suppose two vector fields $$X_1,X_2$$ defined on the 2-manifold $$M=S^2$$ generate $$\mathfrak{X}(M)$$ as a $$C^\infty(M)$$-module. Then by the hairy ball theorem, there exists a point $$p\in S^2$$ where $$X_1(p)=0$$. Then $$T_pM$$ would have to be generated by $$\{0,X_2(p)\}$$, which contradicts the fact that $$T_pM$$ is 2-dimensional. Therefore, $$\mathfrak{X}(M)$$ cannot be generated by the two vector fields $$\{X_1,X_2\}$$. 

On the other hand, for a sufficiently small open set $$U$$, the set $$\mathfrak{X}(U)$$ of vector fields defined on $$U$$ can be regarded as a $$C^\infty(U)$$-module generated by $$m$$ vector fields. For the tangent bundle $$\pi: TM\rightarrow M$$, if we take the open set $$U$$ sufficiently small, we can find a diffeomorphism $$h:U\times\mathbb{R}^m\rightarrow\pi^{-1}(U)$$, so defining vector fields $$X_1,\ldots, X_m$$ by

$$X_1(p)=h(p, e_1),\quad X_2(p)=h(p,e_2),\quad\ldots,\quad X_m(p)=h(p,e_m)\tag{1}$$

these generate $$\mathfrak{X}(U)$$. 

<div class="definition" markdown="1">

<ins id="def4">**Definition 4**</ins> Let a manifold $$M$$ be given, and let $$\dim M=m$$.

- We say that $$X_1,\ldots, X_k$$ are *linearly independent* on a subset $$A$$ of $$M$$ if for each $$p\in A$$, the vectors $$X_1(p),\ldots, X_k(p)$$ in $$T_pM$$ are linearly independent.
- We say that $$X_1,\ldots, X_k$$ *span* the tangent bundle $$TM$$ on a subset $$A$$ of $$M$$ if for each $$p\in A$$, the vectors $$X_1(p),\ldots, X_k(p)$$ in $$T_pM$$ span $$T_pM$$.
- If linearly independent vector fields $$X_1,\ldots, X_k$$ defined on an open set $$U\subseteq M$$ span the tangent bundle, we call them a *local frame* of $$M$$.
- If in the above definition we can take $$U=M$$, we call these vector fields a *global frame* of $$M$$.

</div>

Therefore, from the above discussion we see that a parallelizable manifold has a global frame. Conversely, if a manifold $$M$$ has a global frame $$X_1,\ldots, X_m$$, we can verify that $$M$$ is a parallelizable manifold by defining the map $$TM\rightarrow M\times\mathbb{R}^m$$ by

$$(p,a_1X_1(p)+\cdots+a_mX_m(p))\mapsto (p,a_1e_1+\cdots+a_me_m)$$


## Integral Flow

<div class="definition" markdown="1">

<ins id="def5">**Definition 5**</ins> A $$C^\infty$$ curve $$\sigma$$ on a manifold $$M$$ is called an *integral flow* of $$X\in\mathfrak{X}(M)$$ if $$\sigma'(t)=X(\sigma(t))$$ holds for all $$t$$.

</div>

The following theorem is easily obtained from ordinary differential equations, but we omit its proof since it is beyond our scope. Also, from item 4 onward, the theorem requires a few additional definitions. 

<div class="proposition" markdown="1">

<ins id="thm6">**Theorem 6**</ins> Let $$M$$ be a manifold and $$X\in\mathfrak{X}(M)$$. For each $$p\in M$$ given, there exist suitable constants $$a(p), b(p)$$ (which may include $$\pm\infty$$) and a $$C^\infty$$ curve $$\phi_p: \bigl(a(p),b(p)\bigr)\rightarrow M$$ satisfying the following conditions. 

1. $$0\in \bigl(a(p),b(p)\bigr)$$, $$\phi_p(0)=p$$.
2. $$\phi_p$$ is an integral flow of $$X$$.
3. If $$\mu:(c,d)\rightarrow M$$ is a $$C^\infty$$ function satisfying the above two conditions, then $$(c,d)\subseteq \bigl(a(p),b(p)\bigr)$$, and $$\mu$$ is the same function as $$\phi_p$$ restricted to $$(c,d)$$.
4. For each $$p\in M$$, there exist a suitable open neighborhood $$V$$ and a positive number $$\epsilon$$ such that $$(t,p)\mapsto \phi^{t}(p):=\phi_p(t)$$ is well-defined, and this function is a $$C^\infty$$ map from $$(-\epsilon,\epsilon)\times V$$ to $$M$$.
5. For any $$t$$, the set $$\mathcal{D}_t=\left\{p\in M\mid t\in\bigl(a(p),b(p)\bigr)\right\}$$ is open.
6. $$\bigcup_{t>0}\mathcal{D}_t=M$$.
7. $$\phi^{t}:\mathcal{D}_t\rightarrow\mathcal{D}_{-t}$$ is a diffeomorphism, and its inverse is $$\phi^{-t}$$.
8. The domain of $$\phi^s\circ \phi^t$$ is contained in $$\mathcal{D}_{s+t}$$, and in particular, if $$s$$ and $$t$$ have the same sign, the domain of this function coincides exactly with $$\mathcal{D}_{s+t}$$. Moreover, on this domain, $$\phi^s\circ \phi^t=\phi^{t+s}$$.

</div>

Even though the proof is omitted, we need to fully understand what this theorem means. 

First, items 1 through 3 of the above theorem show that for each $$p\in M$$ there corresponds a unique maximal integral flow, which is uniquely determined except for the choice of the time parameter $$t$$. 

To understand the rest of the theorem, we need some additional explanation of the notation. 

First, the function $$\phi^{t}(p)$$ in condition 4 is defined, as in the definition, by 

$$\phi^{t}(p)=\phi_p(t)$$

and here the initial condition means that we fix $$\phi_p(0)=p$$ at time $$0$$. In this case, we denote the domain of $$\phi^{t}$$ by  

$$\mathcal{D}_t=\left\{p\in M\mid t\in\bigl(a(p),b(p)\bigr)\right\}$$

In other words, we can think of $$\phi^{t}$$ as the function giving the position of the point that was at $$p$$ at time $$0$$ after moving along the integral flow for $$t$$ seconds, and $$\mathcal{D}_t$$ is the set of points for which it is possible to move along the integral flow for $$t$$ seconds. 

Now, except for the technical results 4 and 5, the next result is 6, which is in fact merely a restatement of the existence of integral flows in a different way. Result 7 shows that distinct integral flows do not cross each other. 

Result 8 also has a somewhat technical aspect: if $$s$$ and $$t$$ had opposite signs, say $$s=-2$$ and $$t=1$$, then the domain of $$\phi^s\circ\phi^t$$ would not simply be $$\mathcal{D}_{-1}$$, but rather the set consisting only of those elements of $$\mathcal{D}_{-1}$$ for which it is possible to proceed along $$\phi_p$$ for 1 second.


<div class="definition" markdown="1">

<ins id="def7">**Definition 7**</ins> We say that $$X\in\mathfrak{X}(M)$$ is *complete* if $$\mathcal{D}_t=M$$ for all $$t$$. In this case, the $$\phi^t$$ form a group under $$\circ$$, which is called the *one-parameter group* of $$X$$. 

</div>

If $$X$$ is not complete, there is a slight problem with the domains of the $$\phi^t$$ as above, making it difficult to regard them as a group. A more subtle case is when the original vector field $$X$$ varies with time, but this is not yet our concern, so we skip it.
